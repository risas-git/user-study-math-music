## 🟦 TEST A (Phase 1)

### 1. Module 1: Mixed Operations & Rational Expressions (12 Adaptive Questions)

#### Tier 1 (Easy / Scaffolded)
- **T1_Q1:** Evaluate $5421 - 382 \times 4 + 120 \div 6$  
  - **Options:** `3913` (Correct), `20176`, `3893`  
  - *Misconception Rationale:* BODMAS precedence rule ($382 \times 4 = 1528$, $120 \div 6 = 20 \Rightarrow 5421 - 1528 + 20 = 3913$). In `20176`, operations were performed left-to-right without precedence.
- **T1_Q2:** Evaluate $240 \div (15 - 3) + 18 \times 4$  
  - **Options:** `92` (Correct), `88`, `104`  
  - *Misconception Rationale:* Parentheses first ($240 \div 12 = 20$, $18 \times 4 = 72 \Rightarrow 20 + 72 = 92$). `88` comes from subtracting before completing the addition step.
- **T1_Q3:** Evaluate $\frac{5}{12} + \frac{1}{4} - \frac{1}{3}$  
  - **Options:** `1/3` (Correct), `5/12`, `1/2`  
  - *Misconception Rationale:* Common denominator LCD = $12 \Rightarrow \frac{5+3-4}{12} = \frac{4}{12} = \frac{1}{3}$. In `5/12`, the fraction addition $\frac{1}{4} - \frac{1}{3}$ was miscalculated.

#### Tier 2 (Medium / Baseline Entry)
- **T2_Q1:** Evaluate $\frac{3}{4} + \frac{5}{6} - \frac{2}{3}$  
  - **Options:** `11/12` (Correct), `6/7`, `13/12`  
  - *Misconception Rationale:* LCD = $12 \Rightarrow \frac{9+10-8}{12} = \frac{11}{12}$. `6/7` comes from adding/subtracting numerators and denominators across directly ($\frac{3+5-2}{4+6-3}$).
- **T2_Q2:** Evaluate $\frac{7}{8} - \left(\frac{1}{2} + \frac{1}{4}\right) \times \frac{2}{3}$  
  - **Options:** `3/8` (Correct), `1/4`, `1/2`  
  - *Misconception Rationale:* Parentheses first ($\frac{1}{2} + \frac{1}{4} = \frac{3}{4}$, then $\frac{3}{4} \times \frac{2}{3} = \frac{1}{2}$, then $\frac{7}{8} - \frac{4}{8} = \frac{3}{8}$).
- **T2_Q3:** Evaluate $\frac{4.5 \times 0.6}{0.3} + 2.5$  
  - **Options:** `11.5` (Correct), `9.0`, `14.0`  
  - *Misconception Rationale:* Decimal division precedence $\frac{2.7}{0.3} = 9 \Rightarrow 9 + 2.5 = 11.5$. `9.0` comes from omitting the final $+ 2.5$ addition term.

#### Tier 3 (Hard)
- **T3_Q1:** Simplify $\frac{\frac{4}{9}}{\frac{2}{3} \times \frac{5}{6}}$  
  - **Options:** `4/5` (Correct), `20/81`, `5/4`  
  - *Misconception Rationale:* Invert denominator fraction and multiply: $\frac{4}{9} \div \frac{5}{9} = \frac{4}{9} \times \frac{9}{5} = \frac{4}{5}$. `20/81` comes from multiplying numerators by denominators directly without inverting.
- **T3_Q2:** Simplify $\frac{3}{x+1} - \frac{2}{x-1}$  
  - **Options:** `(x - 5) / ((x+1)(x-1))` (Correct), `1 / ((x+1)(x-1))`, `(x + 1) / ((x+1)(x-1))`  
  - *Misconception Rationale:* Distribute negative sign correctly: $3(x-1) - 2(x+1) = 3x - 3 - 2x - 2 = x - 5$. `1 / ((x+1)(x-1))` comes from ignoring constants in expansion.
- **T3_Q3:** Solve for $x$: $\frac{2}{x+3} + \frac{1}{x} = \frac{5}{x+3}$  
  - **Options:** `x = 3/2` (Correct), `x = 3`, `x = 1`  
  - *Misconception Rationale:* Subtract $\frac{2}{x+3}$ to get $\frac{1}{x} = \frac{3}{x+3} \Rightarrow x+3 = 3x \Rightarrow x = 3/2$. `x = 3` comes from arithmetic error $3x - x = 3x$.

#### Tier 4 (Expert)
- **T4_Q1:** Simplify $\frac{x}{x+2} + \frac{3}{x-1}$  
  - **Options:** `(x^2 + 2x + 6) / ((x+2)(x-1))` (Correct), `(x + 3) / (2x + 1)`, `(x^2 - 4x + 6) / ((x+2)(x-1))`  
  - *Misconception Rationale:* Cross-multiply: $x(x-1) + 3(x+2) = x^2 - x + 3x + 6 = x^2 + 2x + 6$. `(x + 3) / (2x + 1)` comes from adding numerators and denominators across directly.
- **T4_Q2:** Simplify $\frac{\frac{1}{x+h} - \frac{1}{x}}{h}$  
  - **Options:** `-1 / (x(x+h))` (Correct), `1 / (x(x+h))`, `0`  
  - *Misconception Rationale:* Numerator subtraction $x - (x+h) = -h$, retaining the negative sign after dividing by $h$. `0` comes from assuming $\frac{1}{x+h}$ and $\frac{1}{x}$ cancel out.
- **T4_Q3:** Solve for $x$: $\frac{2x}{x-3} + \frac{5}{x+3} = \frac{36}{x^2-9}$  
  - **Options:** `x = 7/2` (Correct), `x = 3` (extraneous), `x = 11/2`  
  - *Misconception Rationale:* Expand $2x(x+3) + 5(x-3) = 36 \Rightarrow 2x^2 + 11x - 51 = 0 \Rightarrow (2x-7)(x+3) = 0 \Rightarrow x = 7/2$. `x = 3` is an extraneous root that makes the denominator equal zero.

### 2. Module 2: Differentiation & Integration (12 Adaptive Questions)

#### Tier 1 (Easy / Scaffolded)
- **M2_T1_Q1:** Find derivative of $y = 3x^4 + 5x^2 - 7x + 2$  
  - **Options:** `12x^3 + 10x - 7` (Correct), `12x^3 + 10x + 2`, `3x^3 + 5x - 7`  
  - *Misconception Rationale:* Power rule $\frac{d}{dx}(x^n) = n x^{n-1}$. Derivative of constant $+2$ is $0$. `12x^3 + 10x + 2` incorrectly retains the constant term $+2$.
- **M2_T1_Q2:** Evaluate $\int (6x^2 + 4x - 5) \, dx$  
  - **Options:** `2x^3 + 2x^2 - 5x + C` (Correct), `12x + 4 + C`, `6x^3 + 4x^2 - 5x + C`  
  - *Misconception Rationale:* Integration rule $\int x^n dx = \frac{x^{n+1}}{n+1} + C \Rightarrow \frac{6x^3}{3} + \frac{4x^2}{2} - 5x = 2x^3 + 2x^2 - 5x + C$. `12x + 4 + C` incorrectly differentiates instead of integrating.
- **M2_T1_Q3:** Find derivative of $y = 4\sqrt{x} + \frac{3}{x}$  
  - **Options:** `2/sqrt(x) - 3/x^2` (Correct), `2/sqrt(x) + 3/x^2`, `2sqrt(x) - 3x^-2`  
  - *Misconception Rationale:* Rewriting terms $4x^{1/2} + 3x^{-1} \Rightarrow 4(\frac{1}{2}x^{-1/2}) + 3(-1x^{-2}) = \frac{2}{\sqrt{x}} - \frac{3}{x^2}$. `2/sqrt(x) + 3/x^2` misses the negative sign when differentiating $x^{-1}$.

#### Tier 2 (Medium / Baseline Entry)
- **M2_T2_Q1:** Find derivative of $y = 5e^{2x} + 3\sin(4x)$  
  - **Options:** `10e^{2x} + 12\cos(4x)` (Correct), `5e^{2x} + 3\cos(4x)`, `10e^{2x} - 12\cos(4x)`  
  - *Misconception Rationale:* Chain Rule $\frac{d}{dx}f(g(x)) = f'(g(x)) \cdot g'(x) \Rightarrow 5(2e^{2x}) + 3(4\cos(4x)) = 10e^{2x} + 12\cos(4x)$. `5e^{2x} + 3\cos(4x)` forgets to multiply by inner derivatives.
- **M2_T2_Q2:** Evaluate $\int_1^3 (3x^2 - 2x) \, dx$  
  - **Options:** `18` (Correct), `20`, `16`  
  - *Misconception Rationale:* Antiderivative $F(x) = x^3 - x^2$. Fundamental Theorem of Calculus $F(3) - F(1) = (27 - 9) - (1 - 1) = 18 - 0 = 18$. `20` comes from evaluation errors at bounds.
- **M2_T2_Q3:** Find tangent line slope for $y = 2x^3 - 5x$ at $x = 2$  
  - **Options:** `19` (Correct), `11`, `24`  
  - *Misconception Rationale:* Derivative function $y' = 6x^2 - 5$. Evaluate at $x = 2 \Rightarrow 6(2^2) - 5 = 24 - 5 = 19$. `11` comes from evaluating the original function $y(2) = 16 - 10 = 6$ instead of $y'(2)$.

#### Tier 3 (Hard)
- **M3_T3_Q1:** Find derivative of $y = x^3 \cdot \ln(x)$  
  - **Options:** `x^2(3ln(x) + 1)` (Correct), `3x^2 * 1/x`, `3x^2 ln(x)`  
  - *Misconception Rationale:* Product Rule $(uv)' = u'v + uv' \Rightarrow (3x^2)\ln(x) + x^3(\frac{1}{x}) = 3x^2\ln(x) + x^2 = x^2(3\ln(x) + 1)$. `3x^2 * 1/x` incorrectly multiplies separate derivatives without applying the Product Rule.
- **M3_T3_Q2:** Evaluate $\int 2x \cos(x^2 + 1) \, dx$  
  - **Options:** `sin(x^2 + 1) + C` (Correct), `x^2 sin(x^2 + 1) + C`, `-sin(x^2 + 1) + C`  
  - *Misconception Rationale:* $u$-substitution $u = x^2+1 \Rightarrow du = 2x\,dx \Rightarrow \int \cos(u)du = \sin(u) + C = \sin(x^2+1) + C$. `-sin(x^2 + 1) + C` confuses integral of cosine ($+\sin$) with derivative ($-\sin$).
- **M3_T3_Q3:** Find derivative of $y = \frac{x^2 + 1}{2x - 3}$  
  - **Options:** `(2x^2 - 6x - 2) / (2x - 3)^2` (Correct), `2x / 2`, `(2x^2 + 6x + 2) / (2x - 3)^2`  
  - *Misconception Rationale:* Quotient Rule $\frac{u'v - uv'}{v^2} \Rightarrow \frac{2x(2x-3) - (x^2+1)(2)}{(2x-3)^2} = \frac{4x^2 - 6x - 2x^2 - 2}{(2x-3)^2} = \frac{2x^2 - 6x - 2}{(2x-3)^2}$. `2x / 2` differentiates numerator and denominator separately.

#### Tier 4 (Expert)
- **M2_T4_Q1:** Evaluate $\int x e^x \, dx$  
  - **Options:** `e^x(x - 1) + C` (Correct), `1/2 x^2 e^x + C`, `e^x(x + 1) + C`  
  - *Misconception Rationale:* Integration by Parts $\int u\,dv = uv - \int v\,du$. Let $u=x, dv=e^x dx \Rightarrow x e^x - \int e^x dx = e^x(x - 1) + C$. `1/2 x^2 e^x + C` integrates factors independently.
- **M2_T4_Q2:** Find derivative of $y = \ln(x^3 + 4x)$  
  - **Options:** `(3x^2 + 4) / (x^3 + 4x)` (Correct), `1 / (x^3 + 4x)`, `(3x^2 + 4) / x`  
  - *Misconception Rationale:* Chain Rule for natural log $\frac{d}{dx}\ln(g(x)) = \frac{g'(x)}{g(x)} = \frac{3x^2 + 4}{x^3 + 4x}$. `1 / (x^3 + 4x)` forgets to multiply by inner derivative $g'(x)$.
- **M2_T4_Q3:** Evaluate $\int_0^{\pi/2} \sin^2(x) \cos(x) \, dx$  
  - **Options:** `1/3` (Correct), `1/2`, `1`  
  - *Misconception Rationale:* $u$-substitution $u = \sin(x) \Rightarrow du = \cos(x)dx$. Bounds change from $0$ to $1 \Rightarrow \int_0^1 u^2 du = [\frac{u^3}{3}]_0^1 = \frac{1}{3}$. `1/2` comes from power integration error.

### 3. Module 3: Linear & Quadratic Equations (12 Adaptive Questions)

#### Tier 1 (Easy / Scaffolded)
- **M3_T1_Q1:** Solve for $x$: $4x - 7 = 17$  
  - **Options:** `x = 6` (Correct), `x = 5`, `x = 4`  
  - *Misconception Rationale:* Add $7$ to both sides: $4x = 24 \Rightarrow x = 6$. `x = 5` comes from subtracting $7$ instead of adding ($4x = 10 \Rightarrow x=2.5$ or arithmetic error $20/4=5$).
- **M3_T1_Q2:** Solve for $x$: $3(x - 2) = 15$  
  - **Options:** `x = 7` (Correct), `x = 5`, `x = 3`  
  - *Misconception Rationale:* Divide by $3$: $x - 2 = 5 \Rightarrow x = 7$. `x = 5` comes from forgetting to add $2$ after dividing ($x = 5$).
- **M3_T1_Q3:** Solve for $x$: $x^2 = 49$  
  - **Options:** `x = +-7` (Correct), `x = 7`, `x = 24.5`  
  - *Misconception Rationale:* Taking square root yields both principal and negative roots $x = \pm 7$. `x = 7` misses the negative root. `x = 24.5` divides by $2$ instead of taking square root.

#### Tier 2 (Medium / Baseline Entry)
- **M3_T2_Q1:** Solve for $x$: $x^2 - 5x + 6 = 0$  
  - **Options:** `x = 2, 3` (Correct), `x = -2, -3`, `x = 1, 6`  
  - *Misconception Rationale:* Factoring $(x-2)(x-3) = 0 \Rightarrow x = +2, +3$. `x = -2, -3` makes a sign error when solving factors ($x-2=0 \Rightarrow x=+2$).
- **M3_T2_Q2:** Solve for $x$: $x^2 + 4x - 12 = 0$  
  - **Options:** `x = 2, -6` (Correct), `x = -2, 6`, `x = 3, -4`  
  - *Misconception Rationale:* Factoring $(x+6)(x-2) = 0 \Rightarrow x = -6, +2$. `x = -2, 6` flips factor signs. `x = 3, -4` uses incorrect factor product pair.
- **M3_T2_Q3:** Solve for $x$: $2x^2 - 8x = 0$  
  - **Options:** `x = 0, 4` (Correct), `x = 4`, `x = 0, -4`  
  - *Misconception Rationale:* Factor out $2x(x-4) = 0 \Rightarrow x = 0, 4$. `x = 4` incorrectly divides by $x$, losing the root $x = 0$.

#### Tier 3 (Hard)
- **M3_T3_Q1:** Solve for $x$: $2x^2 + 7x + 3 = 0$  
  - **Options:** `x = -1/2, -3` (Correct), `x = 1/2, 3`, `x = -1, -3`  
  - *Misconception Rationale:* Quadratic Formula $x = \frac{-b \pm \sqrt{b^2-4ac}}{2a} = \frac{-7 \pm \sqrt{49-24}}{4} = \frac{-7 \pm 5}{4} = -1/2, -3$. `x = 1/2, 3` flips the sign of $-b$.
- **M3_T3_Q2:** Find discriminant of $3x^2 - 5x + 2 = 0$ and state nature of roots  
  - **Options:** `D = 1 (2 real distinct rational roots)` (Correct), `D = -1 (No real roots)`, `D = 49 (2 real roots)`  
  - *Misconception Rationale:* Discriminant $D = b^2 - 4ac = (-5)^2 - 4(3)(2) = 25 - 24 = +1 > 0$. `D = -1` makes a sign subtraction error ($24-25 = -1$).
- **M3_T3_Q3:** Solve for $x$ by completing the square: $x^2 - 6x + 5 = 0$  
  - **Options:** `x = 1, 5` (Correct), `x = -1, -5`, `x = 2, 3`  
  - *Misconception Rationale:* $(x-3)^2 - 9 + 5 = 0 \Rightarrow (x-3)^2 = 4 \Rightarrow x - 3 = \pm 2 \Rightarrow x = 5, 1$. `x = -1, -5` flips signs during isolation.

#### Tier 4 (Expert)
- **M3_T4_Q1:** Solve system: $x + y = 10$ and $xy = 21$  
  - **Options:** `(x,y) = (7,3) or (3,7)` (Correct), `(x,y) = (5,5)`, `(x,y) = (8,2)`  
  - *Misconception Rationale:* Substitute $y = 10-x \Rightarrow x(10-x) = 21 \Rightarrow x^2 - 10x + 21 = 0 \Rightarrow x = 7, 3$. `(x,y) = (5,5)` satisfies sum ($5+5=10$) but violates product ($5 \times 5 = 25 \neq 21$).
- **M3_T4_Q2:** Solve for $x$: $x^4 - 5x^2 + 4 = 0$  
  - **Options:** `x = +-1, +-2` (Correct), `x = 1, 4`, `x = +-1, +-4`  
  - *Misconception Rationale:* Substitute $u = x^2 \Rightarrow u^2 - 5u + 4 = 0 \Rightarrow u = 1, 4 \Rightarrow x = \pm\sqrt{1}, \pm\sqrt{4} = \pm 1, \pm 2$. `x = 1, 4` forgets to take square roots of $u$.
- **M3_T4_Q3:** Solve for $x$: $x + \sqrt{x} - 6 = 0$  
  - **Options:** `x = 4` (Correct), `x = 4, 9`, `x = -3, 2`  
  - *Misconception Rationale:* Substitute $u = \sqrt{x} \Rightarrow u^2 + u - 6 = 0 \Rightarrow (u+3)(u-2) = 0 \Rightarrow u = 2$ ($u=-3$ rejected since $\sqrt{x} \ge 0$) $\Rightarrow x = 2^2 = 4$. `x = 4, 9` includes the extraneous root from $u = -3$.

## 🟪 TEST B (Phase 2 - Isomorphic Parallel Pair)

### 1. Module 1: Mixed Operations & Rational Expressions (12 Isomorphic Questions)

#### Tier 1 (Easy / Scaffolded)
- **T1_Q1:** Evaluate $4356 - 291 \times 3 + 180 \div 9$ $\rightarrow$ **Correct: 3503**  
  - *Misconception Rationale:* BODMAS rule ($291 \times 3 = 873$, $180 \div 9 = 20 \Rightarrow 4356 - 873 + 20 = 3503$).
- **T1_Q2:** Evaluate $360 \div (18 - 6) + 14 \times 5$ $\rightarrow$ **Correct: 100**  
  - *Misconception Rationale:* Parentheses first ($360 \div 12 = 30$, $14 \times 5 = 70 \Rightarrow 30 + 70 = 100$).
- **T1_Q3:** Evaluate $\frac{7}{15} + \frac{1}{5} - \frac{1}{3}$ $\rightarrow$ **Correct: 1/3**  
  - *Misconception Rationale:* LCD = $15 \Rightarrow \frac{7+3-5}{15} = \frac{5}{15} = \frac{1}{3}$.

#### Tier 2 (Medium / Baseline Entry)
- **T2_Q1:** Evaluate $\frac{2}{5} + \frac{3}{4} - \frac{1}{2}$ $\rightarrow$ **Correct: 13/20**  
  - *Misconception Rationale:* LCD = $20 \Rightarrow \frac{8+15-10}{20} = \frac{13}{20}$.
- **T2_Q2:** Evaluate $\frac{5}{6} - \left(\frac{1}{3} + \frac{1}{4}\right) \times \frac{4}{7}$ $\rightarrow$ **Correct: 1/2**  
  - *Misconception Rationale:* Parentheses first ($\frac{7}{12} \times \frac{4}{7} = \frac{1}{3} \Rightarrow \frac{5}{6} - \frac{2}{6} = \frac{3}{6} = \frac{1}{2}$).
- **T2_Q3:** Evaluate $\frac{3.6 \times 0.8}{0.4} + 1.8$ $\rightarrow$ **Correct: 9.0**  
  - *Misconception Rationale:* Decimal division precedence $\frac{2.88}{0.4} = 7.2 \Rightarrow 7.2 + 1.8 = 9.0$.

#### Tier 3 (Hard)
- **T3_Q1:** Simplify $\frac{\frac{3}{8}}{\frac{1}{4} \times \frac{5}{6}}$ $\rightarrow$ **Correct: 9/5**  
  - *Misconception Rationale:* Invert denominator fraction: $\frac{3}{8} \times \frac{24}{5} = \frac{9}{5}$.
- **T3_Q2:** Simplify $\frac{4}{x+2} - \frac{3}{x-2}$ $\rightarrow$ **Correct: (x - 14) / ((x+2)(x-2))**  
  - *Misconception Rationale:* Distribute negative sign correctly: $4(x-2) - 3(x+2) = 4x - 8 - 3x - 6 = x - 14$.
- **T3_Q3:** Solve for $x$: $\frac{3}{x+2} + \frac{1}{x} = \frac{6}{x+2}$ $\rightarrow$ **Correct: x = 2/3**  
  - *Misconception Rationale:* Subtract $\frac{3}{x+2} \Rightarrow \frac{1}{x} = \frac{3}{x+2} \Rightarrow x+2 = 3x \Rightarrow x = 2/3$.

#### Tier 4 (Expert)
- **T4_Q1:** Simplify $\frac{x}{x+3} + \frac{2}{x-1}$ $\rightarrow$ **Correct: (x^2 + 1x + 6) / ((x+3)(x-1))**  
  - *Misconception Rationale:* Cross-multiply: $x(x-1) + 2(x+3) = x^2 - x + 2x + 6 = x^2 + x + 6$.
- **T4_Q2:** Simplify $\frac{\frac{2}{x+h} - \frac{2}{x}}{h}$ $\rightarrow$ **Correct: -2 / (x(x+h))**  
  - *Misconception Rationale:* Numerator subtraction $2(x - (x+h)) = -2h \Rightarrow \frac{-2}{x(x+h)}$.
- **T4_Q3:** Solve for $x$: $\frac{3x}{x-2} + \frac{4}{x+2} = \frac{28}{x^2-4}$ $\rightarrow$ **Correct: x = 5/3**  
  - *Misconception Rationale:* Expand $3x(x+2) + 4(x-2) = 28 \Rightarrow 3x^2 + 10x - 36 = 0 \Rightarrow x = 5/3$ ($x=2$ is an extraneous root).

### 2. Module 2: Differentiation & Integration (12 Isomorphic Questions)

#### Tier 1 (Easy / Scaffolded)
- **M2_T1_Q1:** Find derivative of $y = 4x^4 + 3x^2 - 5x + 9$ $\rightarrow$ **Correct: 16x^3 + 6x - 5**  
  - *Misconception Rationale:* Power rule $\frac{d}{dx}(4x^4+3x^2-5x+9) = 16x^3 + 6x - 5$.
- **M2_T1_Q2:** Evaluate $\int (9x^2 + 6x - 4) \, dx$ $\rightarrow$ **Correct: 3x^3 + 3x^2 - 4x + C**  
  - *Misconception Rationale:* Power integration $\frac{9x^3}{3} + \frac{6x^2}{2} - 4x = 3x^3 + 3x^2 - 4x + C$.
- **M2_T1_Q3:** Find derivative of $y = 6\sqrt{x} + \frac{5}{x}$ $\rightarrow$ **Correct: 3/sqrt(x) - 5/x^2**  
  - *Misconception Rationale:* Derivative of $6x^{1/2} + 5x^{-1} = 3x^{-1/2} - 5x^{-2} = \frac{3}{\sqrt{x}} - \frac{5}{x^2}$.

#### Tier 2 (Medium / Baseline Entry)
- **M2_T2_Q1:** Find derivative of $y = 4e^{3x} + 2\sin(5x)$ $\rightarrow$ **Correct: 12e^{3x} + 10\cos(5x)**  
  - *Misconception Rationale:* Chain Rule $4(3e^{3x}) + 2(5\cos(5x)) = 12e^{3x} + 10\cos(5x)$.
- **M2_T2_Q2:** Evaluate $\int_1^2 (6x^2 - 4x) \, dx$ $\rightarrow$ **Correct: 8**  
  - *Misconception Rationale:* Antiderivative $F(x) = 2x^3 - 2x^2 \Rightarrow F(2) - F(1) = (16-8) - (2-2) = 8$.
- **M2_T2_Q3:** Find tangent line slope for $y = 3x^3 - 4x$ at $x = 2$ $\rightarrow$ **Correct: 32**  
  - *Misconception Rationale:* Derivative $y' = 9x^2 - 4 \Rightarrow y'(2) = 36 - 4 = 32$.

#### Tier 3 (Hard)
- **M2_T3_Q1:** Find derivative of $y = x^4 \cdot \ln(x)$ $\rightarrow$ **Correct: x^3(4ln(x) + 1)**  
  - *Misconception Rationale:* Product Rule $(4x^3)\ln(x) + x^4(\frac{1}{x}) = 4x^3\ln(x) + x^3 = x^3(4\ln(x) + 1)$.
- **M2_T3_Q2:** Evaluate $\int 3x^2 \sin(x^3 + 2) \, dx$ $\rightarrow$ **Correct: -cos(x^3 + 2) + C**  
  - *Misconception Rationale:* $u$-substitution $u = x^3+2 \Rightarrow du = 3x^2 dx \Rightarrow \int \sin(u)du = -\cos(u) + C$.
- **M2_T3_Q3:** Find derivative of $y = \frac{x^2 + 3}{3x - 2}$ $\rightarrow$ **Correct: (3x^2 - 4x - 9) / (3x - 2)^2**  
  - *Misconception Rationale:* Quotient Rule $\frac{2x(3x-2) - 3(x^2+3)}{(3x-2)^2} = \frac{6x^2 - 4x - 3x^2 - 9}{(3x-2)^2} = \frac{3x^2 - 4x - 9}{(3x-2)^2}$.

#### Tier 4 (Expert)
- **M2_T4_Q1:** Evaluate $\int 2x e^{2x} \, dx$ $\rightarrow$ **Correct: e^{2x}(x - 1/2) + C**  
  - *Misconception Rationale:* Integration by Parts $u=x, dv=2e^{2x}dx \Rightarrow x e^{2x} - \int e^{2x}dx = e^{2x}(x - 1/2) + C$.
- **M2_T4_Q2:** Find derivative of $y = \ln(x^4 + 5x)$ $\rightarrow$ **Correct: (4x^3 + 5) / (x^4 + 5x)**  
  - *Misconception Rationale:* Chain Rule for natural log $\frac{g'(x)}{g(x)} = \frac{4x^3 + 5}{x^4 + 5x}$.
- **M2_T4_Q3:** Evaluate $\int_0^{\pi/2} \cos^2(x) \sin(x) \, dx$ $\rightarrow$ **Correct: 1/3**  
  - *Misconception Rationale:* $u$-substitution $u = \cos(x) \Rightarrow du = -\sin(x)dx$. Bounds flip from $1$ to $0 \Rightarrow \int_0^1 u^2 du = \frac{1}{3}$.

### 3. Module 3: Linear & Quadratic Equations (12 Isomorphic Questions)

#### Tier 1 (Easy / Scaffolded)
- **M3_T1_Q1:** Solve for $x$: $5x - 8 = 22$ $\rightarrow$ **Correct: x = 6**  
  - *Misconception Rationale:* Add $8$ to both sides: $5x = 30 \Rightarrow x = 6$.
- **M3_T1_Q2:** Solve for $x$: $4(x - 3) = 16$ $\rightarrow$ **Correct: x = 7**  
  - *Misconception Rationale:* Divide by $4$: $x - 3 = 4 \Rightarrow x = 7$.
- **M3_T1_Q3:** Solve for $x$: $x^2 = 64$ $\rightarrow$ **Correct: x = +-8**  
  - *Misconception Rationale:* Square root yields both positive and negative roots $x = \pm 8$.

#### Tier 2 (Medium / Baseline Entry)
- **M3_T2_Q1:** Solve for $x$: $x^2 - 7x + 12 = 0$ $\rightarrow$ **Correct: x = 3, 4**  
  - *Misconception Rationale:* Factoring $(x-3)(x-4) = 0 \Rightarrow x = +3, +4$.
- **M3_T2_Q2:** Solve for $x$: $x^2 + 5x - 14 = 0$ $\rightarrow$ **Correct: x = 2, -7**  
  - *Misconception Rationale:* Factoring $(x+7)(x-2) = 0 \Rightarrow x = -7, +2$.
- **M3_T2_Q3:** Solve for $x$: $3x^2 - 12x = 0$ $\rightarrow$ **Correct: x = 0, 4**  
  - *Misconception Rationale:* Factor out $3x(x-4) = 0 \Rightarrow x = 0, 4$.

#### Tier 3 (Hard)
- **M3_T3_Q1:** Solve for $x$: $2x^2 + 9x + 4 = 0$ $\rightarrow$ **Correct: x = -1/2, -4**  
  - *Misconception Rationale:* Quadratic Formula $x = \frac{-9 \pm \sqrt{81-32}}{4} = \frac{-9 \pm 7}{4} = -1/2, -4$.
- **M3_T3_Q2:** Find discriminant of $2x^2 - 7x + 3 = 0$ $\rightarrow$ **Correct: D = 25 (2 real distinct rational roots)**  
  - *Misconception Rationale:* Discriminant $D = b^2 - 4ac = (-7)^2 - 4(2)(3) = 49 - 24 = +25 > 0$.
- **M3_T3_Q3:** Solve for $x$ by completing square: $x^2 - 8x + 12 = 0$ $\rightarrow$ **Correct: x = 2, 6**  
  - *Misconception Rationale:* $(x-4)^2 = 4 \Rightarrow x - 4 = \pm 2 \Rightarrow x = 6, 2$.

#### Tier 4 (Expert)
- **M3_T4_Q1:** Solve system: $x + y = 12$ and $xy = 35$ $\rightarrow$ **Correct: (x,y) = (7,5) or (5,7)**  
  - *Misconception Rationale:* Substitute $y = 12-x \Rightarrow x(12-x) = 35 \Rightarrow x^2 - 12x + 35 = 0 \Rightarrow x = 7, 5$.
- **M3_T4_Q2:** Solve for $x$: $x^4 - 10x^2 + 9 = 0$ $\rightarrow$ **Correct: x = +-1, +-3**  
  - *Misconception Rationale:* Substitute $u = x^2 \Rightarrow u^2 - 10u + 9 = 0 \Rightarrow u = 1, 9 \Rightarrow x = \pm 1, \pm 3$.
- **M3_T4_Q3:** Solve for $x$: $x + 2\sqrt{x} - 8 = 0$ $\rightarrow$ **Correct: x = 4**  
  - *Misconception Rationale:* Substitute $u = \sqrt{x} \Rightarrow u^2 + 2u - 8 = 0 \Rightarrow (u+4)(u-2) = 0 \Rightarrow u = 2 \Rightarrow x = 4$.
