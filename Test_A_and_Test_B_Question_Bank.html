🟦 TEST A (Phase 1)

1. Module 1: Mixed Operations & Rational Expressions

Tier 1 (Easy / Scaffolded)

T1_Q1: Evaluate $5421 - 382 \times 4 + 120 \div 6$

Options: 3913 (Correct), 20176, 3893

💡 Misconception Rationale: BODMAS precedence rule ($382 \times 4 = 1528$, $120 \div 6 = 20 \Rightarrow 5421 - 1528 + 20 = 3913$). 20176 ignores operational precedence.

T1_Q2: Evaluate $240 \div (15 - 3) + 18 \times 4$

Options: 92 (Correct), 88, 104

💡 Misconception Rationale: Parentheses first ($240 \div 12 = 20$, $18 \times 4 = 72 \Rightarrow 20 + 72 = 92$).

T1_Q3: Evaluate $\frac{5}{12} + \frac{1}{4} - \frac{1}{3}$

Options: 1/3 (Correct), 5/12, 1/2

💡 Misconception Rationale: LCD = $12 \Rightarrow \frac{5+3-4}{12} = \frac{4}{12} = \frac{1}{3}$.
Tier 2 (Medium / Baseline Entry)

T2_Q1: Evaluate $\frac{3}{4} + \frac{5}{6} - \frac{2}{3}$

Options: 11/12 (Correct), 6/7, 13/12

💡 Misconception Rationale: LCD = $12 \Rightarrow \frac{9+10-8}{12} = \frac{11}{12}$. 6/7 comes from adding/subtracting across numerators/denominators directly.

T2_Q2: Evaluate $\frac{7}{8} - \left(\frac{1}{2} + \frac{1}{4}\right) \times \frac{2}{3}$

Options: 3/8 (Correct), 1/4, 1/2

💡 Misconception Rationale: Parentheses first ($\frac{3}{4} \times \frac{2}{3} = \frac{1}{2} \Rightarrow \frac{7}{8} - \frac{4}{8} = \frac{3}{8}$).

T2_Q3: Evaluate $\frac{4.5 \times 0.6}{0.3} + 2.5$

Options: 11.5 (Correct), 9.0, 14.0

💡 Misconception Rationale: Decimal division precedence $\frac{2.7}{0.3} = 9 \Rightarrow 9 + 2.5 = 11.5$.
Tier 3 (Hard)

T3_Q1: Simplify $\frac{\frac{4}{9}}{\frac{2}{3} \times \frac{5}{6}}$

Options: 4/5 (Correct), 20/81, 5/4

💡 Misconception Rationale: Invert denominator fraction: $\frac{4}{9} \times \frac{9}{5} = \frac{4}{5}$.

T3_Q2: Simplify $\frac{3}{x+1} - \frac{2}{x-1}$

Options: (x - 5) / ((x+1)(x-1)) (Correct), 1 / ((x+1)(x-1)), (x + 1) / ((x+1)(x-1))

💡 Misconception Rationale: Distribute negative sign: $3(x-1) - 2(x+1) = 3x - 3 - 2x - 2 = x - 5$.

T3_Q3: Solve for $x$: $\frac{2}{x+3} + \frac{1}{x} = \frac{5}{x+3}$

Options: x = 3/2 (Correct), x = 3, x = 1

💡 Misconception Rationale: $\frac{1}{x} = \frac{3}{x+3} \Rightarrow x+3 = 3x \Rightarrow x = 3/2$.
Tier 4 (Expert)

T4_Q1: Simplify $\frac{x}{x+2} + \frac{3}{x-1}$

Options: (x^2 + 2x + 6) / ((x+2)(x-1)) (Correct), (x + 3) / (2x + 1), (x^2 - 4x + 6) / ((x+2)(x-1))

💡 Misconception Rationale: Cross-multiply: $x(x-1) + 3(x+2) = x^2 + 2x + 6$.

T4_Q2: Simplify $\frac{\frac{1}{x+h} - \frac{1}{x}}{h}$

Options: -1 / (x(x+h)) (Correct), 1 / (x(x+h)), 0

💡 Misconception Rationale: Numerator subtraction $x - (x+h) = -h \Rightarrow \frac{-1}{x(x+h)}$.

T4_Q3: Solve for $x$: $\frac{2x}{x-3} + \frac{5}{x+3} = \frac{36}{x^2-9}$

Options: x = 7/2 (Correct), x = 3 (extraneous), x = 11/2

💡 Misconception Rationale: Expand $2x(x+3) + 5(x-3) = 36 \Rightarrow x = 7/2$ ($x=3$ is extraneous).

2. Module 2: Differentiation & Integration

Tier 1 (Easy / Scaffolded)

M2_T1_Q1: Find derivative of $y = 3x^4 + 5x^2 - 7x + 2$

Options: 12x^3 + 10x - 7 (Correct), 12x^3 + 10x + 2, 3x^3 + 5x - 7

💡 Misconception Rationale: Power rule $\frac{d}{dx}(x^n) = n x^{n-1}$. Constant derivative is 0.

M2_T1_Q2: Evaluate $\int (6x^2 + 4x - 5) \, dx$

Options: 2x^3 + 2x^2 - 5x + C (Correct), 12x + 4 + C, 6x^3 + 4x^2 - 5x + C

💡 Misconception Rationale: Integration rule $\int x^n dx = \frac{x^{n+1}}{n+1} + C$.

M2_T1_Q3: Find derivative of $y = 4\sqrt{x} + \frac{3}{x}$

Options: 2/sqrt(x) - 3/x^2 (Correct), 2/sqrt(x) + 3/x^2, 2sqrt(x) - 3x^-2

💡 Misconception Rationale: $4(\frac{1}{2}x^{-1/2}) + 3(-1x^{-2}) = \frac{2}{\sqrt{x}} - \frac{3}{x^2}$.
Tier 2 (Medium / Baseline Entry)

M2_T2_Q1: Find derivative of $y = 5e^{2x} + 3\sin(4x)$

Options: 10e^{2x} + 12\cos(4x) (Correct), 5e^{2x} + 3\cos(4x), 10e^{2x} - 12\cos(4x)

💡 Misconception Rationale: Chain Rule: multiply by inner derivatives 2 and 4.

M2_T2_Q2: Evaluate $\int_1^3 (3x^2 - 2x) \, dx$

Options: 18 (Correct), 20, 16

💡 Misconception Rationale: $F(3) - F(1) = (27-9) - (1-1) = 18$.

M2_T2_Q3: Tangent line slope for $y = 2x^3 - 5x$ at $x = 2$

Options: 19 (Correct), 11, 24

💡 Misconception Rationale: Evaluate derivative $y' = 6x^2 - 5 \Rightarrow 6(4) - 5 = 19$.
Tier 3 (Hard)

M2_T3_Q1: Derivative of $y = x^3 \cdot \ln(x)$

Options: x^2(3ln(x) + 1) (Correct), 3x^2 * 1/x, 3x^2 ln(x)

💡 Misconception Rationale: Product Rule $(uv)' = u'v + uv' \Rightarrow x^2(3\ln(x)+1)$.

M2_T3_Q2: Evaluate $\int 2x \cos(x^2 + 1) \, dx$

Options: sin(x^2 + 1) + C (Correct), x^2 sin(x^2 + 1) + C, -sin(x^2 + 1) + C

💡 Misconception Rationale: $u$-substitution $u=x^2+1 \Rightarrow \int \cos(u)du = \sin(x^2+1)+C$.

M2_T3_Q3: Derivative of $y = \frac{x^2 + 1}{2x - 3}$

Options: (2x^2 - 6x - 2) / (2x - 3)^2 (Correct), 2x / 2, (2x^2 + 6x + 2) / (2x - 3)^2

💡 Misconception Rationale: Quotient Rule $\frac{u'v - uv'}{v^2} = \frac{2x^2 - 6x - 2}{(2x-3)^2}$.
Tier 4 (Expert)

M2_T4_Q1: Evaluate $\int x e^x \, dx$

Options: e^x(x - 1) + C (Correct), 1/2 x^2 e^x + C, e^x(x + 1) + C

💡 Misconception Rationale: Integration by Parts $\int u\,dv = uv - \int v\,du \Rightarrow e^x(x-1)+C$.

M2_T4_Q2: Derivative of $y = \ln(x^3 + 4x)$

Options: (3x^2 + 4) / (x^3 + 4x) (Correct), 1 / (x^3 + 4x), (3x^2 + 4) / x

💡 Misconception Rationale: Chain Rule for natural log $\frac{g'(x)}{g(x)} = \frac{3x^2+4}{x^3+4x}$.

M2_T4_Q3: Evaluate $\int_0^{\pi/2} \sin^2(x) \cos(x) \, dx$

Options: 1/3 (Correct), 1/2, 1

💡 Misconception Rationale: $u$-sub $u=\sin(x) \Rightarrow \int_0^1 u^2 du = [\frac{u^3}{3}]_0^1 = 1/3$.

3. Module 3: Linear & Quadratic Equations

Tier 1 (Easy / Scaffolded)

M3_T1_Q1: Solve for $x$: $4x - 7 = 17$

Options: x = 6 (Correct), x = 5, x = 4

💡 Misconception Rationale: Add 7 to both sides: $4x = 24 \Rightarrow x = 6$.

M3_T1_Q2: Solve for $x$: $3(x - 2) = 15$

Options: x = 7 (Correct), x = 5, x = 3

💡 Misconception Rationale: Divide by 3: $x-2 = 5 \Rightarrow x = 7$.

M3_T1_Q3: Solve for $x$: $x^2 = 49$

Options: x = +-7 (Correct), x = 7, x = 24.5

💡 Misconception Rationale: Square root yields both positive and negative roots $x = \pm 7$.
Tier 2 (Medium / Baseline Entry)

M3_T2_Q1: Solve for $x$: $x^2 - 5x + 6 = 0$

Options: x = 2, 3 (Correct), x = -2, -3, x = 1, 6

💡 Misconception Rationale: Factoring $(x-2)(x-3) = 0 \Rightarrow x = +2, +3$.

M3_T2_Q2: Solve for $x$: $x^2 + 4x - 12 = 0$

Options: x = 2, -6 (Correct), x = -2, 6, x = 3, -4

💡 Misconception Rationale: Factoring $(x+6)(x-2) = 0 \Rightarrow x = -6, +2$.

M3_T2_Q3: Solve for $x$: $2x^2 - 8x = 0$

Options: x = 0, 4 (Correct), x = 4, x = 0, -4

💡 Misconception Rationale: Factor out $2x(x-4) = 0 \Rightarrow x = 0, 4$.
Tier 3 (Hard)

M3_T3_Q1: Solve for $x$: $2x^2 + 7x + 3 = 0$

Options: x = -1/2, -3 (Correct), x = 1/2, 3, x = -1, -3

💡 Misconception Rationale: Quadratic Formula $\frac{-7 \pm 5}{4} = -1/2, -3$.

M3_T3_Q2: Discriminant of $3x^2 - 5x + 2 = 0$

Options: D = 1 (2 real distinct rational roots) (Correct), D = -1 (No real roots), D = 49 (2 real roots)

💡 Misconception Rationale: $D = b^2 - 4ac = 25 - 24 = +1 > 0$.

M3_T3_Q3: Completing the square: $x^2 - 6x + 5 = 0$

Options: x = 1, 5 (Correct), x = -1, -5, x = 2, 3

💡 Misconception Rationale: $(x-3)^2 = 4 \Rightarrow x - 3 = \pm 2 \Rightarrow x = 5, 1$.
Tier 4 (Expert)

M3_T4_Q1: Solve system: $x + y = 10$ and $xy = 21$

Options: (x,y) = (7,3) or (3,7) (Correct), (x,y) = (5,5), (x,y) = (8,2)

💡 Misconception Rationale: $x(10-x) = 21 \Rightarrow x^2 - 10x + 21 = 0 \Rightarrow x = 7, 3$.

M3_T4_Q2: Solve for $x$: $x^4 - 5x^2 + 4 = 0$

Options: x = +-1, +-2 (Correct), x = 1, 4, x = +-1, +-4

💡 Misconception Rationale: Substitute $u = x^2 \Rightarrow u = 1, 4 \Rightarrow x = \pm 1, \pm 2$.

M3_T4_Q3: Solve for $x$: $x + \sqrt{x} - 6 = 0$

Options: x = 4 (Correct), x = 4, 9, x = -3, 2

💡 Misconception Rationale: Substitute $u = \sqrt{x} \Rightarrow (u+3)(u-2) = 0 \Rightarrow u = 2 \Rightarrow x = 4$.

🟪 TEST B (Phase 2 - Isomorphic Pair)

1. Module 1: Mixed Operations & Rational Expressions

T1_Q1: Evaluate $4356 - 291 \times 3 + 180 \div 9$ $\rightarrow$ 3503

💡 Misconception Rationale: BODMAS precedence rule ($291 \times 3 = 873$, $180 \div 9 = 20 \Rightarrow 4356 - 873 + 20 = 3503$).

T1_Q2: Evaluate $360 \div (18 - 6) + 14 \times 5$ $\rightarrow$ 100

💡 Misconception Rationale: Parentheses first ($360 \div 12 = 30$, $14 \times 5 = 70 \Rightarrow 30 + 70 = 100$).

T1_Q3: Evaluate $\frac{7}{15} + \frac{1}{5} - \frac{1}{3}$ $\rightarrow$ 1/3

💡 Misconception Rationale: LCD = $15 \Rightarrow \frac{7+3-5}{15} = \frac{5}{15} = \frac{1}{3}$.

T2_Q1: Evaluate $\frac{2}{5} + \frac{3}{4} - \frac{1}{2}$ $\rightarrow$ 13/20

💡 Misconception Rationale: LCD = $20 \Rightarrow \frac{8+15-10}{20} = \frac{13}{20}$.

T2_Q2: Evaluate $\frac{5}{6} - \left(\frac{1}{3} + \frac{1}{4}\right) \times \frac{4}{7}$ $\rightarrow$ 1/2

💡 Misconception Rationale: Parentheses first ($\frac{7}{12} \times \frac{4}{7} = \frac{1}{3} \Rightarrow \frac{5}{6} - \frac{2}{6} = \frac{3}{6} = \frac{1}{2}$).

T2_Q3: Evaluate $\frac{3.6 \times 0.8}{0.4} + 1.8$ $\rightarrow$ 9.0

💡 Misconception Rationale: Decimal division precedence $\frac{2.88}{0.4} = 7.2 \Rightarrow 7.2 + 1.8 = 9.0$.

2. Module 2: Differentiation & Integration

M2_T1_Q1: Find derivative of $y = 4x^4 + 3x^2 - 5x + 9$ $\rightarrow$ 16x^3 + 6x - 5

💡 Misconception Rationale: Power rule $\frac{d}{dx}(4x^4+3x^2-5x+9) = 16x^3 + 6x - 5$.

M2_T1_Q2: Evaluate $\int (9x^2 + 6x - 4) \, dx$ $\rightarrow$ 3x^3 + 3x^2 - 4x + C

💡 Misconception Rationale: Power integration $\frac{9x^3}{3} + \frac{6x^2}{2} - 4x = 3x^3 + 3x^2 - 4x + C$.

M2_T2_Q1: Find derivative of $y = 4e^{3x} + 2\sin(5x)$ $\rightarrow$ 12e^{3x} + 10\cos(5x)

💡 Misconception Rationale: Chain Rule $4(3e^{3x}) + 2(5\cos(5x)) = 12e^{3x} + 10\cos(5x)$.

M2_T2_Q2: Evaluate $\int_1^2 (6x^2 - 4x) \, dx$ $\rightarrow$ 8

💡 Misconception Rationale: Antiderivative $F(x) = 2x^3 - 2x^2 \Rightarrow F(2) - F(1) = 8$.

3. Module 3: Linear & Quadratic Equations

M3_T1_Q1: Solve for $x$: $5x - 8 = 22$ $\rightarrow$ x = 6

💡 Misconception Rationale: Add 8 to both sides: $5x = 30 \Rightarrow x = 6$.

M3_T2_Q1: Solve for $x$: $x^2 - 7x + 12 = 0$ $\rightarrow$ x = 3, 4

💡 Misconception Rationale: Factoring $(x-3)(x-4) = 0 \Rightarrow x = +3, +4$.

M3_T3_Q1: Solve for $x$: $2x^2 + 9x + 4 = 0$ $\rightarrow$ x = -1/2, -4

💡 Misconception Rationale: Quadratic Formula $x = \frac{-9 \pm \sqrt{81-32}}{4} = -1/2, -4$.

M3_T4_Q1: Solve system: $x + y = 12$ and $xy = 35$ $\rightarrow$ (x,y) = (7,5) or (5,7)

💡 Misconception Rationale: Substitute $y = 12-x \Rightarrow x^2 - 12x + 35 = 0 \Rightarrow x = 7, 5$.
