Module 3: Linear & Quadratic Equations

This module tests your ability to solve linear, quadratic, non-linear system, and biquadratic equations. Review the fundamental rules below before starting the exercise.

1. Linear Isolation & Square Roots

• Isolate $x$ by adding/subtracting constants and dividing by coefficients: $ax + b = c \Rightarrow x = \frac{c - b}{a}$.
• For simple square root equations $x^2 = k$, remember there are two roots: $x = \pm \sqrt{k}$.

2. Quadratic Factoring

Factor quadratic expressions $x^2 + bx + c = 0$ into binomials $(x - p)(x - q) = 0$.
Set each factor to zero: $x - p = 0 \Rightarrow x = p$ and $x - q = 0 \Rightarrow x = q$.

3. Quadratic Formula & Discriminant ($D$)

For any quadratic equation $ax^2 + bx + c = 0$: $$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$$ • Discriminant $D = b^2 - 4ac$:
— If $D > 0$: 2 real distinct roots.
— If $D = 0$: 1 real repeated root.
— If $D < 0$: No real roots.

4. Systems of Equations & Variable Substitution

To solve non-linear systems like $x + y = S$ and $xy = P$, substitute $y = S - x$ into the product equation to form a quadratic equation $x^2 - Sx + P = 0$.
